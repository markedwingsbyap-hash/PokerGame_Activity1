POKER GAME CONSOLE APPLICATION BY YAP, MARK EDWIND BSCS 3-2

This is a simple Console-based Poker game built using C#.

The goal of the game is to test the luck of each user by providing them with virtual coins, and see if they can win good poker hands.


Mechanics & Rules of the Game:

Starting Balance: 

  The user/player begins with 500 coins.

Betting
  
  Each round has a fixed cost of 100 coins/entrance fee to play the round. This is automatically deducted from your balance at the start of the round.

Gameplay:

  The system generates a deck of 52 cards and deals 5 random cards to the player.

  The system analyzes the hand of the user to see if it matches any "Winning Combinations."

Bankrupt:
  
  If your balance falls below 100 coins, he/she is declared "Bankrupt," and the game ends.

History/Scoreboard: 

  The program displays a record of your previous rounds at the top of the screen, including your starting balance per round, the cards dealt, the resulting hand, and your updated balance.

Exit:

  After every round, the system will provide the user with an option to exit the game or continue playing.
